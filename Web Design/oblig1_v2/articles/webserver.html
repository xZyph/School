Webservers

A webserver is defined as a machine that with it's hardware and software is set up to serve content over the HTTP protocol, to the World Wide Web.

There are loads of software that all can do this, and you can even make your own if you want. Mostly though, people tend to use one of the big, free and very available applications that already exist.

Some examples of webserver-applications:

  • IIS (Microsoft)
  • nginx (NGINX, Inc)
  • Apache (Apache)
  • GWS (Google)
