Installation of Wordpress

This is a short and graphic demonstration of how you can install Wordpress. Be sure to hover over the images, as some of them magically will start to move. The sizes do not match exactly, and because of time issues I haven't had time to go over and match everything.

Download the latest version of Wordpress from their website.

After the download we'll need to unzip the compressed files.

Copy all the files to the host folder. This might be c:\xampp\htdocs, /var/www or any other path depending on webserver software and operating system.

Install and set up a SQL database of your own choice. In the picture I've installed MySQL and use the HeidiSQL application to graphically manage it. After the SQL is set up, it's time to create a database for our Wordpress.

Wordpress will also need a database user, so let's create one of these as well.

Wordpress should now be ready to install, so open the host in your browser and select the preferred language.

Enter all the required connection information for your database, along with the newly created user and database.

Lastly, enter the login information you would like to use on your freshly installed Wordpress.
