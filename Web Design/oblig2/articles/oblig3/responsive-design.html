Responsive Design

Media Queries are what we use to define rulesets for different situations. We use these to set custom rulesets for printers, different devices and is capable of much more.

Media Types are the actual device the CSS is meant for. Amongst others we have important media types like screen and print.

With both CSS Flexbox and CSS Grid we are able to achieve Flexible Sizing, which allows elements to easily take up fractions of space. The flexible sizing allows us to create websites that naturally resize and fit the media you are viewing it on.

Font Sizing is also a very underestimated tool for building responsive websites. By using the em unit to set paddings and margins, all of the spacing will depend on the root font size. This means that by just changing the font size in the media query, you will also change the spacing. Please note that I sadly haven't implemented this functionality on this website, but perhaps the next one?
